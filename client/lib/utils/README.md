# Utils Module

Модуль общих утилит и вспомогательных функций, предназначенных для обеспечения повторного использования кода, стандартизации операций и упрощения разработки в рамках клиентской части приложения Telegram Bot Builder.

Этот модуль содержит набор функций, которые решают часто встречающиеся задачи, такие как работа с пользовательскими данными, форматирование, валидация, а также генерация кода для ботов. Файлы в этом модуле организованы по функциональному назначению, что упрощает поиск и использование необходимых инструментов.

Utils Module играет ключевую роль в обеспечении согласованности и надежности кода, позволяя сосредоточиться на логике приложения, а не на реализации базовых операций.

## Файлы

- `user-utils.ts` - Утилиты для работы с пользователями
- `utils.ts` - Общие утилитные функции
- `generateReplaceVariablesFunction.ts` - Генерация функции замены переменных
- `generateInitUserVariablesFunction.ts` - Генерация функции инициализации пользовательских переменных
- `generateUniversalVariableReplacement.ts` - Генерация универсальной функции замены переменных
- `checkAutoTransition.ts` - Функция проверки автопереходов между узлами

## Использование

```typescript
import { 
  initUserVariables,
  validateUser,
  formatUserData 
} from './utils';

// Инициализация переменных пользователя
const userVars = initUserVariables(userId);

// Валидация пользовательских данных
const isValid = validateUser(userData);
```

## Категории утилит

### Работа с пользователями
- Инициализация пользовательских переменных
- Валидация пользовательских данных
- Форматирование информации о пользователях
- Управление сессиями пользователей

### Общие утилиты
- Функции для работы со строками
- Утилиты для работы с датами
- Валидация данных
- Форматирование вывода
