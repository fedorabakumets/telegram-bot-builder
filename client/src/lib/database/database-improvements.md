# Этапы улучшения папки `client/src/lib/database`

## Общие требования

- [ ] Все новые файлы и изменения кода должны содержать JSDoc комментарии на русском языке с @fileoverview
- [ ] Каждый файл с генерируемым кодом должен иметь комментарий @fileoverview, аналогично log_message.ts
- [ ] Генерируемый код должен автоматически получать комментарии с использованием processCodeWithAutoComments, как реализовано в log_message.ts
