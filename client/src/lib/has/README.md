# Папка Has

Эта папка содержит утилиты для проверки наличия различных возможностей в боте.

## Содержимое

- `hasConditionalButtons` - проверяет наличие условных кнопок
- `hasMediaNodes` - проверяет наличие узлов с медиа
- `hasInputCollection` - проверяет наличие сбора ввода
- `hasInlineButtons` - проверяет наличие инлайн-кнопок
- `hasAutoTransitions` - проверяет наличие автоматических переходов
- `hasMultiSelectNodes` - проверяет наличие узлов с множественным выбором
- `hasLocationFeatures` - проверяет наличие функций местоположения
