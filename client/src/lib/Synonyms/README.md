# Synonyms Module

Модуль для генерации обработчиков синонимов команд и сообщений.

## Файлы

- `generateMessageSynonymHandler.ts` - Генерация обработчиков синонимов сообщений
- `generateSynonymHandler.ts` - Генерация общих обработчиков синонимов

## Использование

```typescript
import { 
  generateMessageSynonymHandler,
  generateSynonymHandler 
} from './Synonyms';

// Генерация обработчика синонимов сообщений
const messageHandler = generateMessageSynonymHandler(synonymGroups);

// Генерация общего обработчика синонимов
const synonymHandler = generateSynonymHandler(commandSynonyms);
```

## Типы синонимов

### Синонимы команд
- Альтернативные названия команд
- Сокращения команд
- Команды на разных языках

### Синонимы сообщений
- Различные варианты ответов пользователей
- Обработка опечаток и вариаций
- Поддержка естественного языка

## Особенности

- Нечеткое сопоставление текста
- Поддержка регулярных выражений
- Приоритизация синонимов
- Интеграция с основными обработчиками
- Логирование совпадений для аналитики

## Применение

- Улучшение пользовательского опыта
- Обработка естественного языка
- Поддержка многоязычности
- Снижение количества непонятых команд
