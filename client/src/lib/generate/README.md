# Папка Generate

Эта папка содержит утилиты для генерации различных частей кода бота.

## Содержимое

- `generateDatabaseCode` - генерация кода для работы с базой данных
- `generateNodeNavigation` - генерация кода для навигации между узлами
- `generateUtf8EncodingCode` - генерация кода для установки UTF-8 кодировки
- `generateSafeEditOrSendCode` - генерация безопасного кода для редактирования или отправки сообщений
- `generateBasicBotSetupCode` - генерация базового кода настройки бота
- `generateGroupsConfiguration` - генерация кода конфигурации групп
- `generateMessageLoggingCode` - генерация кода для логирования сообщений
- `generateUtilityFunctions` - генерация вспомогательных функций
