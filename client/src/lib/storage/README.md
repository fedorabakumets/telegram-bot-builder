# Storage Module

Модуль для работы с локальным хранилищем данных.

## Файлы

- `local-storage.ts` - Утилиты для работы с localStorage браузера

## Использование

```typescript
import { localStorage } from './storage';

// Сохранение данных
localStorage.setItem('bot-config', JSON.stringify(config));

// Получение данных
const config = JSON.parse(localStorage.getItem('bot-config') || '{}');
```

## Особенности

- Типизированные методы для работы с localStorage
- Обработка ошибок сериализации/десериализации
- Утилиты для работы с различными типами данных
- Кэширование настроек и конфигураций

## Применение

- Сохранение пользовательских настроек
- Кэширование данных между сессиями
- Хранение временных данных редактора
- Настройки отладки и логирования
