# Validate Module

Модуль для валидации структуры бота и его компонентов.

## Файлы

- `validateBotStructure.ts` - Валидация общей структуры бота
- `validateCommand.ts` - Валидация команд и их параметров

## Использование

```typescript
import { 
  validateBotStructure,
  validateCommand 
} from './validate';

// Валидация структуры бота
const structureErrors = validateBotStructure(botData);

// Валидация команды
const commandErrors = validateCommand(commandNode);
```

## Типы валидации

### Структура бота
- Проверка целостности узлов и связей
- Валидация обязательных полей
- Проверка циклических зависимостей
- Валидация типов данных

### Команды
- Проверка синтаксиса команд
- Валидация параметров команд
- Проверка уникальности команд
- Валидация прав доступа

## Возвращаемые ошибки

```typescript
interface ValidationError {
  type: 'error' | 'warning';
  message: string;
  nodeId?: string;
  field?: string;
}
```

## Особенности

- Подробные сообщения об ошибках
- Поддержка предупреждений и критических ошибок
- Контекстная информация для исправления
- Интеграция с системой типов TypeScript
- Валидация на разных уровнях (узел, бот, проект)

## Применение

- Проверка данных перед генерацией кода
- Валидация пользовательского ввода
- Отладка структуры бота
- Обеспечение качества генерируемого кода
