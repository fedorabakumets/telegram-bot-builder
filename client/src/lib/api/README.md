# API Module

Модуль для работы с API и HTTP запросами.

## Файлы

- `queryClient.ts` - Конфигурация и утилиты для работы с API клиентом

## Использование

```typescript
import { queryClient } from './api';
```

Этот модуль предоставляет централизованную конфигурацию для работы с внешними API и управления HTTP запросами в приложении.
