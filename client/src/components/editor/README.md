# Компоненты редактора

## Terminal

Компонент терминала для отображения вывода питон-скрипта бота.

### Установка зависимостей

Для корректной работы компонента Terminal необходимо установить следующую зависимость:

```bash
npm install ansi-to-react
```

Если используются типы TypeScript:

```bash
npm install @types/ansi-to-react
```

### Использование

Компонент предоставляет интерфейс для отображения вывода (stdout/stderr) запущенного питон-скрипта бота в формате, похожем на терминал VS Code.
