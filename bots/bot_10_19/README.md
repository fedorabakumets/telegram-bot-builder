## Установка

1. Клонируйте или скачайте файлы проекта
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Создайте файл `.env` и добавьте настройки:
   ```
   BOT_TOKEN=your_bot_token_here
   DATABASE_URL=postgresql://user:password@localhost:5432/bot_db
   ```

4. Настройте базу данных PostgreSQL (опционально):
   - Создайте базу данных PostgreSQL
   - Обновите DATABASE_URL в .env файле
   - Бот автоматически создаст необходимые таблицы при запуске
   - Если БД недоступна, бот будет использовать локальное хранилище

5. Запустите бота:
   ```bash
   python bot.py
   ```
