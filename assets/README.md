# Assets

Статические ресурсы проекта - изображения, иконки и другие медиафайлы.

## Структура

### `images/` - Изображения проекта
- `bot_added_ui_visible.png` - скриншот интерфейса с добавленным ботом
- `cycle1_after_return.png` - диаграмма цикла после возврата
- `new_bot_added.png` - скриншот добавления нового бота

## Типы ресурсов

### Скриншоты интерфейса
Изображения, демонстрирующие работу приложения:
- **UI скриншоты** - показывают различные состояния интерфейса
- **Процессы** - иллюстрируют пошаговые действия пользователя
- **Результаты** - демонстрируют конечный результат работы

### Диаграммы и схемы
Визуальные материалы для документации:
- **Диаграммы потоков** - показывают логику работы системы
- **Архитектурные схемы** - структура приложения
- **UML диаграммы** - модели данных и процессов

## Использование в проекте

### В README файлах
```markdown
![Bot Interface](assets/images/bot_added_ui_visible.png)
```

### В документации
```markdown
<img src="assets/images/new_bot_added.png" alt="New Bot Added" width="600">
```

### В компонентах React
```typescript
import botImage from '@/assets/images/bot_added_ui_visible.png';

function BotPreview() {
  return <img src={botImage} alt="Bot Preview" />;
}
```

## Требования к файлам

### Изображения
- **Формат**: PNG для скриншотов, JPG для фотографий
- **Размер**: максимум 2 МБ на файл
- **Разрешение**: оптимально 1920x1080 для скриншотов
- **Качество**: высокое качество для документации

### Именование файлов
- Используйте описательные имена на английском языке
- Разделяйте слова подчеркиванием: `bot_added_ui_visible.png`
- Включайте контекст: `cycle1_after_return.png`

### Оптимизация
- Сжимайте изображения для веб-использования
- Используйте WebP для современных браузеров
- Создавайте версии разных размеров при необходимости

## Организация файлов

### По типу контента
```
assets/
├── images/           # Растровые изображения
├── icons/           # Иконки и символы
├── logos/           # Логотипы проекта
├── screenshots/     # Скриншоты интерфейса
└── diagrams/        # Диаграммы и схемы
```

### По разделам проекта
```
assets/
├── auth/            # Ресурсы для аутентификации
├── bot-builder/     # Ресурсы конструктора ботов
├── dashboard/       # Ресурсы панели управления
└── landing/         # Ресурсы главной страницы
```

## Инструменты для работы

### Создание скриншотов
- **Windows**: Snipping Tool, Greenshot
- **macOS**: Screenshot utility (Cmd+Shift+4)
- **Linux**: GNOME Screenshot, Flameshot

### Редактирование изображений
- **Простое редактирование**: GIMP, Paint.NET
- **Профессиональное**: Adobe Photoshop, Figma
- **Оптимизация**: TinyPNG, ImageOptim

### Создание диаграмм
- **Онлайн**: Draw.io, Lucidchart
- **Локально**: Visio, OmniGraffle
- **Код**: Mermaid, PlantUML
