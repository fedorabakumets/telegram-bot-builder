# .kiro

Папка конфигурации и данных для Kiro AI Assistant.

## Структура

### `specs/` - Спецификации проекта
Формализованные спецификации для разработки функций:
- `bot-generator-refactor/` - спецификация рефакторинга генератора ботов
- `component-architecture-refactor/` - спецификация рефакторинга архитектуры компонентов

## Назначение

### Спецификации (Specs)
Kiro использует спецификации для:
- **Структурированной разработки** сложных функций
- **Итеративного планирования** и реализации
- **Документирования** процесса разработки
- **Контроля качества** и соответствия требованиям

### Процесс работы со спецификациями
1. **Создание спецификации** - формализация требований
2. **Планирование** - разбивка на этапы
3. **Итеративная разработка** - пошаговая реализация
4. **Тестирование и валидация** - проверка соответствия
5. **Документирование** - фиксация результатов

## Структура спецификации

### Основные разделы
- **Обзор** - краткое описание задачи
- **Требования** - детальные требования
- **Архитектура** - техническое решение
- **План реализации** - этапы разработки
- **Тестирование** - критерии приемки

### Пример структуры
```
specs/
├── feature-name/
│   ├── overview.md          # Обзор функции
│   ├── requirements.md      # Требования
│   ├── architecture.md      # Архитектурное решение
│   ├── implementation.md    # План реализации
│   ├── testing.md          # План тестирования
│   └── progress.md         # Прогресс выполнения
```
