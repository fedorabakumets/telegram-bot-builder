# 🧮 Узел «Переменные» — все режимы (modes)

## Текущее название: "🧮 Переменные" (set_variable)

### Рекомендация по переименованию: **🧮 Вычисления**

Отражает суть — узел не просто "устанавливает переменные", а вычисляет, форматирует, ищет, генерирует, трансформирует.

| Вариант | Плюсы | Минусы |
|---------|-------|--------|
| 🧮 Вычисления | Точно отражает суть, короткое | Не покрывает строковые операции |
| ⚙️ Обработка данных | Универсальное | Слишком общее |
| 🔧 Трансформации | Как в ETL, профессионально | Непонятно новичкам |
| 📐 Формулы | Математический акцент | Не покрывает lookup/random |

**Итог:** «🧮 Вычисления» — лучший баланс между точностью и понятностью.

### 1. text — Подстановка строки/переменных

**Описание:** Устанавливает переменную в текстовое значение с подстановкой других переменных через `{var}`.

**JSON:**
```json
{
  "variable": "greeting",
  "value": "Привет, {pilot.name}! Баланс: {credits} кр.",
  "mode": "text"
}
```

**Примеры:**
| # | value | Переменные | Результат | Описание |
|---|-------|-----------|-----------|----------|
| 1 | `Привет, {name}!` | name=Алекс | `Привет, Алекс!` | Приветствие |
| 2 | `Баланс: {credits} кр.` | credits=5000 | `Баланс: 5000 кр.` | Показ баланса |
| 3 | `{ship} → {planet}` | ship=Сокол, planet=Марс | `Сокол → Марс` | Маршрут |
| 4 | `Заказ #{order_id} оформлен` | order_id=1847 | `Заказ #1847 оформлен` | Подтверждение заказа |
| 5 | `{rank}: {name} [{clan}]` | rank=Капитан, name=Иван, clan=ЗВ | `Капитан: Иван [ЗВ]` | Профиль игрока |
| 6 | `Осталось {lives} жизней` | lives=3 | `Осталось 3 жизней` | Игровой статус |

**Генерация Python:**
```python
user_data[user_id]["greeting"] = replace_variables_in_text("Привет, {pilot.name}! Баланс: {credits} кр.", _vars)
```

**Применение:** Формирование сообщений, профилей, статусов, подписей к фото.

### 2. expression — Арифметическое выражение

**Описание:** Вычисляет математическое выражение с подстановкой переменных.

**JSON:**
```json
{
  "variable": "total_price",
  "value": "{quantity} * {price_per_unit} - {discount}",
  "mode": "expression"
}
```

**Примеры:**
| # | value | Переменные | Результат | Описание |
|---|-------|-----------|-----------|----------|
| 1 | `{credits} + 100` | credits=500 | `600` | Начисление бонуса |
| 2 | `{hp} - {damage}` | hp=100, damage=35 | `65` | Урон в бою |
| 3 | `{price} * {quantity}` | price=250, quantity=3 | `750` | Стоимость покупки |
| 4 | `{fuel} - {distance} * 2` | fuel=1000, distance=150 | `700` | Расход топлива |
| 5 | `({base_xp} + {bonus_xp}) * {multiplier}` | base_xp=50, bonus_xp=20, multiplier=2 | `140` | Опыт с множителем |
| 6 | `{balance} * 90 // 100` | balance=1500 | `1350` | Комиссия 10% |

**Генерация Python:**
```python
_expr = replace_variables_in_text("{quantity} * {price_per_unit} - {discount}", _vars)
user_data[user_id]["total_price"] = str(eval(_expr))
```

**Применение:** Расчёт цен, урона, опыта, комиссий, любая математика.

### 9. lookup — Поиск в таблице

**Описание:** Ищет значение в таблице (массив объектов) по ключу. Аналог VLOOKUP в Excel.

**JSON:**
```json
{
  "variable": "planet_name",
  "value": "{planet_id}",
  "mode": "lookup",
  "lookupTable": [
    {"key": "1", "value": "Земля"},
    {"key": "2", "value": "Марс"},
    {"key": "3", "value": "Юпитер"}
  ],
  "defaultValue": "Неизвестная"
}
```

**Примеры:**
| # | Ввод (key) | Таблица | Результат | Описание |
|---|-----------|---------|-----------|----------|
| 1 | `1` | planets | `Земля` | ID → название планеты |
| 2 | `vip` | ranks | `👑 VIP` | Ранг → эмодзи |
| 3 | `ru` | langs | `Русский` | Код → язык |
| 4 | `999` | planets | `Неизвестная` | Не найдено → default |
| 5 | `iron` | resources | `⛏ Железо` | Ресурс → отображение |
| 6 | `admin` | roles | `Администратор` | Роль → название |

**Генерация Python:**
```python
_key = replace_variables_in_text("{planet_id}", _vars)
_table = [{"key": "1", "value": "Земля"}, {"key": "2", "value": "Марс"}, {"key": "3", "value": "Юпитер"}]
_result = "Неизвестная"
for _row in _table:
    if _row["key"] == _key:
        _result = _row["value"]
        break
user_data[user_id]["planet_name"] = _result
```

**Применение:** Справочники, маппинг ID→название, локализация, ранги.

### 12. json_format — Массив → строка

**Описание:** Преобразует JSON-массив в читаемую строку с нумерацией или разделителем.

**JSON:**
```json
{
  "variable": "inventory_text",
  "value": "{inventory}",
  "mode": "json_format",
  "formatTemplate": "{index}. {item}",
  "separator": "\n"
}
```

**Примеры:**
| # | Массив | Шаблон | Результат | Описание |
|---|--------|--------|-----------|----------|
| 1 | `["меч", "щит"]` | `{index}. {item}` | `1. меч\n2. щит` | Нумерованный список |
| 2 | `["Марс", "Юпитер"]` | `🪐 {item}` | `🪐 Марс\n🪐 Юпитер` | С эмодзи |
| 3 | `["A", "B", "C"]` | `{item}` (sep=`, `) | `A, B, C` | Через запятую |
| 4 | `["iron", "gold"]` | `• {item}` | `• iron\n• gold` | Маркированный список |
| 5 | `["task1"]` | `✅ {item}` | `✅ task1` | Один элемент |
| 6 | `[]` | `{item}` | `(пусто)` | Пустой массив |

**Генерация Python:**
```python
import json
_arr = json.loads(replace_variables_in_text("{inventory}", _vars))
_lines = [f"{i+1}. {item}" for i, item in enumerate(_arr)]
user_data[user_id]["inventory_text"] = "\n".join(_lines) if _lines else "(пусто)"
```

**Применение:** Отображение инвентаря, списков задач, корзины, истории.

### 13. weighted_random — Взвешенный рандом

**Описание:** Случайный выбор элемента с указанными весами (процентами). Сумма весов должна быть 100.

**JSON:**
```json
{
  "variable": "event",
  "mode": "weighted_random",
  "weightedItems": [
    {"value": "ничего", "weight": 60},
    {"value": "пираты", "weight": 25},
    {"value": "артефакт", "weight": 10},
    {"value": "джекпот", "weight": 5}
  ]
}
```

**Примеры:**
| # | Items + Weights | Возможный результат | Описание |
|---|-----------------|---------------------|----------|
| 1 | ничего:60, пираты:25, артефакт:10, джекпот:5 | `пираты` | Событие при перелёте |
| 2 | common:70, rare:20, epic:8, legendary:2 | `rare` | Дроп из сундука |
| 3 | win:45, lose:45, draw:10 | `win` | Исход дуэли |
| 4 | 10%:30, 20%:40, 50%:20, 100%:10 | `20%` | Размер скидки |
| 5 | iron:40, copper:30, gold:20, crystal:10 | `copper` | Добыча ресурса |
| 6 | miss:50, hit:35, crit:15 | `hit` | Результат атаки |

**Генерация Python:**
```python
import random as _rnd
_items = [("ничего", 60), ("пираты", 25), ("артефакт", 10), ("джекпот", 5)]
_roll = _rnd.randint(1, 100)
_cumulative = 0
_result = _items[-1][0]
for _item, _weight in _items:
    _cumulative += _weight
    if _roll <= _cumulative:
        _result = _item
        break
user_data[user_id]["event"] = _result
```

**Применение:** Лут-таблицы, события, дроп, гача-механики, исходы боёв.

### 15. daily_random — Детерминированный рандом по дате

**Описание:** Генерирует число, одинаковое для всех пользователей в один день. Разные seed → разные числа.

**JSON:**
```json
{
  "variable": "iron_mult",
  "value": "iron",
  "mode": "daily_random",
  "minValue": "70",
  "maxValue": "130"
}
```

**Примеры:**
| # | seed | min | max | Результат (пример) | Описание |
|---|------|-----|-----|---------------------|----------|
| 1 | `iron` | 70 | 130 | `94` | Множитель цены железа |
| 2 | `gold` | 80 | 150 | `127` | Множитель цены золота |
| 3 | `bonus` | 1 | 3 | `2` | Множитель опыта дня |
| 4 | `weather` | 0 | 4 | `3` | Погода на планете (индекс) |
| 5 | `shop_discount` | 5 | 30 | `18` | Скидка дня (%) |
| 6 | `daily_quest` | 1 | 10 | `7` | Номер квеста дня |

**Генерация Python:**
```python
import hashlib, time
_seed = f"{time.strftime('%Y-%m-%d')}_{replace_variables_in_text('iron', _vars)}"
_hash = int(hashlib.md5(_seed.encode()).hexdigest()[:8], 16)
_min, _max = 70, 130
_result = _hash % (_max - _min + 1) + _min
```

### 30. pluralize — Склонение слова по числу

**Описание:** Выбирает правильную форму слова в зависимости от числа. Для русского языка: 3 формы (1 кредит, 2 кредита, 5 кредитов).

**JSON:**
```json
{
  "variable": "credits_word",
  "value": "{credits}",
  "mode": "pluralize",
  "forms": ["кредит", "кредита", "кредитов"]
}
```

**Правило (русский):** forms[0] для 1, 21, 31...; forms[1] для 2-4, 22-24...; forms[2] для 5-20, 25-30...

**Примеры:**
| # | Число | forms | Результат | Описание |
|---|-------|-------|-----------|----------|
| 1 | `1` | `["кредит", "кредита", "кредитов"]` | `кредит` | Один |
| 2 | `3` | `["кредит", "кредита", "кредитов"]` | `кредита` | Два-четыре |
| 3 | `15` | `["кредит", "кредита", "кредитов"]` | `кредитов` | Пять и более |
| 4 | `21` | `["день", "дня", "дней"]` | `день` | 21 — как 1 |
| 5 | `0` | `["жизнь", "жизни", "жизней"]` | `жизней` | Ноль |
| 6 | `102` | `["предмет", "предмета", "предметов"]` | `предмета` | 102 — как 2 |

**Генерация Python:**
```python
_n = int(replace_variables_in_text("{credits}", _vars))
_forms = ["кредит", "кредита", "кредитов"]
if _n % 10 == 1 and _n % 100 != 11:
    _result = _forms[0]
elif 2 <= _n % 10 <= 4 and not (12 <= _n % 100 <= 14):
    _result = _forms[1]
else:
    _result = _forms[2]
user_data[user_id]["credits_word"] = _result
```

**Применение:** Любой текст с числами: "3 кредита", "1 день", "5 предметов", "22 игрока".
