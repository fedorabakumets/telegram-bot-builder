# app_settings

## app_settings

Таблица настроек приложения в формате ключ-значение.  
Используется для хранения глобальных параметров конфигурации,  
например флага завершения мастера первоначальной настройки.  
[CI test] проверка автосинка docs/database.

### Columns

| Name | Type | Default | Nullable | Children | Parents | Comment |
|------|------|---------|----------|----------|---------|---------|
| **key** | text | - | NO | - | - | Уникальный ключ настройки |
| value | text | - | NO | - | - | Значение настройки |
| updated_at | timestamp | `now()` | YES | - | - | Дата последнего обновления |
