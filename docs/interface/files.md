# 📁 Вкладка «Файлы»

«Файлы» собирает в одном месте все медиафайлы проекта — фотографии, видео, документы и другие вложения, которые проходят через бота. Список разбит по страницам: внизу виден счётчик в формате «N файлов • стр. X/Y», а стрелки перелистывают страницы по 50 файлов. Здесь можно отфильтровать файлы по источнику и типу, выбрать нужные и удалить лишние, а также скопировать идентификатор любого файла, чтобы повторно использовать его в боте.

## Источник файлов

В верхней панели находится переключатель источника — он показывает, откуда пришёл файл:

| Источник | Что показывает |
|----------|----------------|
| Входящие | Файлы, которые присылали боту пользователи |
| Исходящие | Файлы, которые бот отправлял пользователям |
| Загруженные | Файлы, добавленные в проект вручную |

За раз показывается один источник. При переключении список обновляется, страница сбрасывается на первую, а сделанный выбор файлов снимается.

## Фильтр по типу медиа

Рядом с переключателем источника находится фильтр по типу медиа. По умолчанию выбран вариант «Все типы» — показываются файлы любого вида. Чтобы оставить только нужные, выберите один из типов:

- **Все типы** — без ограничения по виду файла.
- **Фото** — изображения.
- **Видео** — видеоролики.
- **GIF** — анимации.
- **Аудио** — музыкальные и звуковые файлы.
- **Голосовое** — голосовые сообщения.
- **Кружок** — круглые видеосообщения.
- **Документы** — файлы-вложения (например, PDF или архивы).
- **Стикеры** — стикеры.

При смене типа список сразу пересобирается и снова открывается на первой странице.

## Просмотр файлов

Каждый файл занимает строку с превью, названием, расширением, типом, размером и датой загрузки. Если у файла есть превью (фото, видео, стикер или анимация), миниатюру можно открыть на весь экран и пролистать соседние файлы стрелками. Файл можно скачать кнопкой «Скачать».

## Выбор и удаление

Слева от каждого файла стоит галочка для выбора. Галочка в заголовке списка отмечает сразу все файлы на странице. Как только выбран хотя бы один файл, внизу появляется плашка со счётчиком «Выбрано: N» и двумя кнопками:

- **Снять выбор** — сбрасывает все отметки.
- **Удалить (N)** — удаляет выбранные файлы, где N — их количество.

Удаление затрагивает только отмеченные файлы из текущего источника, поэтому перед нажатием стоит проверить, что выбраны именно нужные записи.

## Идентификатор файла

У каждого файла есть идентификатор `file_id` — короткая строка, по которой Telegram узнаёт этот файл. Идентификатор показан в отдельной колонке: щёлкните по нему или нажмите кнопку копирования рядом, чтобы скопировать значение в буфер обмена. Тот же идентификатор можно затем вставить в сценарий бота, чтобы отправлять готовый файл повторно, не загружая его заново.

Если у файла есть обложка (например, превью видео), её идентификатор показан в колонке «Обл.» и копируется отдельной кнопкой.

## Выбор бота и проекта

В шапке вкладки можно переключить проект и выбрать бота (токен), для которого показываются файлы — так удобно смотреть файлы разных ботов одного проекта. Кнопка «Обновить» перезагружает список, если файлы появились или изменились с момента последнего открытия вкладки.
