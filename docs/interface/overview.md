# 🖥️ Интерфейс

Конструктор устроен как набор вкладок. Каждая вкладка отвечает за свою часть работы: одна — за сборку логики бота, другие — за запуск, пользователей, рассылки, статистику и данные. Вы переключаетесь между вкладками по ходу проекта: сначала собираете сценарий в «Редакторе», затем запускаете бота, смотрите диалоги, делаете рассылки и анализируете результаты.

Этот раздел описывает, что делает каждая вкладка и как ей пользоваться. Подробно расписана главная вкладка — **«Редактор»**, а также все остальные вкладки приложения.

## Вкладки приложения

| Вкладка | Зачем нужна |
|---------|-------------|
| [Редактор](editor.md) | Главная рабочая область — визуальный конструктор бота: ноды, холст, соединения |
| [Бот](bot.md) | Запуск и остановка бота, статус подключения, токен и основные настройки |
| [Терминал](terminal.md) | Логи работы бота в реальном времени и вывод сообщений системы |
| [Пользователи](users.md) | Список людей, которые пользуются ботом, и информация о них |
| [Диалоги](dialogs.md) | История переписки бота с пользователями |
| [Рассылки](broadcasts/overview.md) | Массовая отправка сообщений выбранным группам пользователей |
| [Аналитика](analytics.md) | Статистика по командам, сообщениям и активности |
| [Таблицы](tables/overview.md) | Хранилище данных бота в виде таблиц |
| [Файлы](files.md) | Файлы проекта и загруженные материалы |
| [Код](code.md) | Сгенерированный код бота по собранному сценарию |

## С чего начать

Если вы только открыли проект — начните с вкладки **[Редактор](editor.md)**. Там вы собираете логику бота из нод: добавляете триггеры (на что бот реагирует) и действия (что бот делает), соединяете их между собой и настраиваете каждый блок.
