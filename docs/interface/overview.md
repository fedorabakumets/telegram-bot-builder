# 🖥️ Интерфейс

Конструктор устроен как набор вкладок. Каждая вкладка отвечает за свою часть работы: одна — за сборку логики бота, другие — за запуск, пользователей, рассылки, статистику и данные. Вы переключаетесь между вкладками по ходу проекта: сначала собираете сценарий в «Редакторе», затем запускаете бота, смотрите диалоги, делаете рассылки и анализируете результаты.

Этот раздел описывает, что делает каждая вкладка и как ей пользоваться. Сейчас подробно расписана главная вкладка — **«Редактор»**. Описания остальных вкладок будут добавлены позже.

## Вкладки приложения

| Вкладка | Зачем нужна |
|---------|-------------|
| [Редактор](editor.md) | Главная рабочая область — визуальный конструктор бота: ноды, холст, соединения |
| Бот | Запуск и остановка бота, статус подключения, токен и основные настройки |
| Терминал | Логи работы бота в реальном времени и вывод сообщений системы |
| Пользователи | Список людей, которые пользуются ботом, и информация о них |
| Диалоги | История переписки бота с пользователями |
| Рассылки | Массовая отправка сообщений выбранным группам пользователей |
| Аналитика | Статистика по командам, сообщениям и активности |
| Таблицы | Хранилище данных бота в виде таблиц |
| Файлы | Файлы проекта и загруженные материалы |
| Код | Сгенерированный код бота по собранному сценарию |

> Описания вкладок «Бот», «Терминал», «Пользователи», «Диалоги», «Рассылки», «Аналитика», «Таблицы», «Файлы» и «Код» будут добавлены в следующих обновлениях документации.

## С чего начать

Если вы только открыли проект — начните с вкладки **[Редактор](editor.md)**. Там вы собираете логику бота из нод: добавляете триггеры (на что бот реагирует) и действия (что бот делает), соединяете их между собой и настраиваете каждый блок.
