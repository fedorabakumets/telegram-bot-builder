# Редактор

«Редактор» — главная рабочая область конструктора. Здесь вы собираете бота визуально: добавляете блоки (ноды), расставляете их на холсте, соединяете между собой и настраиваете каждый. Это та вкладка, где рождается вся логика бота — от реакции на команду `/start` до сложных сценариев с условиями и рассылками.

Работа похожа на сборку схемы: берёте нужный блок из палитры слева, ставите на холст, тянете стрелку к следующему блоку — и шаг за шагом выстраиваете путь, по которому бот будет вести пользователя.

## Из чего состоит вкладка

| Область | Где находится | Для чего |
|---------|---------------|----------|
| Сайдбар с нодами | слева | Палитра блоков, которые можно добавить на холст |
| Холст | в центре | Рабочее поле, где вы собираете и соединяете ноды |
| Панель свойств | справа | Настройки выбранной ноды |
| Тулбар | сверху | Кнопки управления: масштаб, история, сохранение и др. |
| Панель листов | снизу | Переключение между листами сценария |

Любую из боковых панелей можно скрыть, чтобы освободить место для холста — за это отвечают кнопки в тулбаре.

## Холст

Холст — это поле, на котором живут все ноды вашего бота. Что с ним можно делать:

- **Добавить ноду** — перетащить её из сайдбара на холст.
- **Переместить ноду** — захватить мышью и перетащить в нужное место.
- **Выделить ноду** — кликнуть по ней. Выделенную ноду можно настраивать и удалять.
- **Удалить ноду** — выделить и нажать `Delete`.
- **Масштаб (зум)** — приблизить или отдалить картинку от 1% до 200% (по умолчанию 100%).
- **Панорамирование** — двигать холст, чтобы увидеть другие его части.
- **Отмена и повтор** — любое действие можно отменить и вернуть обратно.
- **Вписать содержимое** — автоматически подогнать масштаб так, чтобы все ноды поместились на экране, или сфокусироваться на конкретном узле.

> Мини-карты холста в редакторе нет — для навигации используйте панорамирование и кнопку «уместить всё».

## Соединения между нодами

Соединение — это стрелка, которая задаёт порядок: после какой ноды бот переходит к следующей. Чтобы создать соединение, потяните мышью от **выходного порта** (кружок справа от ноды) к нужной ноде.

Цвет соединения подсказывает его тип:

| Цвет | Тип | Что означает |
|------|-----|--------------|
| 🟡 Жёлтый | trigger-next | Переход из триггера команды к первой ноде сценария |
| 🟢 Зелёный | auto-transition | Автоматический переход к следующей ноде |
| 🔵 Синий | button-goto | Переход по нажатию кнопки |
| 🟠 Янтарный | keyboard-link | Привязка клавиатуры к сообщению |
| 🟣 Фиолетовый | input-target | Переход после того, как пользователь что-то ввёл |

**Чтобы удалить соединение** — кликните по линии.

## Сайдбар с нодами

Слева находится палитра блоков. Ноды сгруппированы по категориям, каждую категорию можно свернуть; рядом с названием показан счётчик нод внутри.

Категории:

- **Сообщения** — отправка текста, медиа, ожидание ответа, редактирование и удаление сообщений.
- **Клавиатура** — кнопки и реакции на их нажатие.
- **Группы** — работа в группах: темы форума, исключение участников.
- **Автоматизация** — запуск по расписанию.
- **Интеграции** — HTTP-запросы, базы данных, таблицы, условия, переменные, циклы.
- **Юзербот** — действия от имени пользовательского аккаунта.

### Как добавить ноду

- **Перетащить** карточку ноды из сайдбара на холст.
- **Нажать «+»** на карточке (кнопка появляется при наведении) — нода добавится в центр холста.

### Пресеты команд

В секции **«Команды»** есть готовые пресеты. Один такой пресет создаёт сразу пару блоков — триггер команды и сообщение, — чтобы не собирать связку вручную.

### Поиск

В сайдбаре есть **поиск узлов по проекту** — удобно, когда нод много и нужно быстро найти нужную.

Кроме палитры, у сайдбара есть вкладки **«Проекты»** и **«Компоненты»**, а также **импорт** сценария из JSON или Python.

## Тулбар

Панель кнопок сверху холста:

| Кнопка | Что делает |
|--------|-----------|
| Масштаб | Уменьшить, увеличить, сбросить, уместить всё, выбрать процент |
| Отменить / повторить | Откат и возврат действий |
| История действий | Просмотр истории, откат выбранных действий |
| Сохранить | Сохранить проект |
| Авто-раскладка | Автоматически расставить узлы по холсту |
| Копировать / вставить | Перенос нод между проектами |
| Показать / скрыть панели | Управление видимостью: заголовок, сайдбар, холст, свойства |
| Справка | Подсказка по горячим клавишам |
| Холст / JSON | Переключение между визуальным холстом и текстовым видом проекта |

> Отдельной кнопки «превью» нет — посмотреть проект целиком можно в режиме **JSON**.

## Панель свойств

Панель свойств открывается справа, когда вы **кликаете по ноде**. В ней собраны все настройки выбранного блока — текст сообщения, кнопки, условия и прочее, в зависимости от типа ноды.

На мобильных устройствах настройки открываются отдельной панелью.

## Листы сценария

Проект можно разбить на несколько **листов** — это как страницы в одной тетради. Удобно, когда сценарий большой: разные части логики держат на разных листах. Панель листов находится снизу.

- **Добавить лист** — кнопка «+». Новому листу даётся имя автоматически («Лист N»).
- **Переименовать** — двойной клик по вкладке листа.
- **Дублировать** — кнопка дублирования листа.
- **Удалить** — кнопка удаления (доступна, когда листов больше одного).
- **Прокрутка** — если листов много, появляются стрелки прокрутки.

На мобильных устройствах листы переключаются **свайпом**.

## Горячие клавиши

### Масштаб

| Клавиши | Действие |
|---------|----------|
| `Ctrl` + `+` | Увеличить |
| `Ctrl` + `-` | Уменьшить |
| `Ctrl` + `0` | Сбросить масштаб |
| `Ctrl` + `1` | Уместить всё |
| `Ctrl` + `2` или `A` | Авто-уместить |

### Редактирование

| Клавиши | Действие |
|---------|----------|
| `Delete` | Удалить узел |
| `Ctrl` + `C` или `D` | Дублировать |
| `Ctrl` + `Shift` + `C` | Копировать в буфер |
| `Ctrl` + `V` | Вставить |

> Горячие клавиши работают и с русской раскладкой (буквы `С`, `М`, `В`).

### История

| Клавиши | Действие |
|---------|----------|
| `Ctrl` + `Z` | Отменить |
| `Ctrl` + `Y` или `Ctrl` + `Shift` + `Z` | Повторить |

### Навигация

| Клавиши | Действие |
|---------|----------|
| `Ctrl` + колесо мыши | Масштаб |
| `Alt` + левая кнопка мыши | Панорамирование |

### Файл

| Клавиши | Действие |
|---------|----------|
| `Ctrl` + `S` | Сохранить |
