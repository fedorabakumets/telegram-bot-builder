# 💬 Вкладка «Диалоги»

Вкладка «Диалоги» — это переписка вашего бота с людьми. Слева находится список диалогов, справа открывается выбранная беседа: вы читаете историю сообщений и можете ответить от имени бота, как в обычном мессенджере.

## История переписки

Когда вы выбираете пользователя из списка слева, справа открывается вся история переписки с ним. Сообщения расположены по времени: входящие (от пользователя) и исходящие (от бота) видно сразу.

История обновляется в реальном времени. Если пользователь пишет боту новое сообщение, оно появляется в открытом диалоге автоматически — обновлять страницу вручную не нужно.

## Ответ от имени бота

Внизу открытого диалога есть поле «Введите сообщение...». Наберите в нём текст и нажмите кнопку «Отправить» — сообщение уйдёт пользователю от имени бота и появится в переписке.

Уже отправленные ботом сообщения можно поправить:

- **Редактировать** — изменить текст ранее отправленного сообщения.
- **Удалить** — убрать отправленное ботом сообщение из переписки.

## Прикрепление медиафайлов

К сообщению можно приложить медиа — фото, видео, аудио или документ. Для этого рядом с полем ввода есть кнопка-скрепка: она открывает выбор файла.

Если у вас уже есть идентификатор файла в Telegram (file_id), его можно вставить напрямую через кнопку «Добавить Telegram file_id». Это удобно, когда нужный файл уже загружен в Telegram и вы хотите переслать его без повторной загрузки.

## Переключение между пользователями

Не выходя из панели диалога, можно переходить от одной беседы к другой: просто выберите другого пользователя в списке слева. Открытая справа переписка сменится на историю выбранного собеседника.

## Диалоги с группами и каналами

Кроме личных бесед, в списке встречаются диалоги с группами и каналами — они показываются как отдельный случай. Под названием беседы отображается её тип:

- **Группа** — обычная группа.
- **Супергруппа** — расширенная группа.
- **Канал** — канал.

В таких диалогах рядом с сообщениями видно отправителя — кто именно из участников написал сообщение.

## Открытие диалога из других мест

Диалог с пользователем не обязательно искать вручную в списке. Его можно открыть из других мест приложения — например, из вкладки «Пользователи» по кнопке «Открыть в Диалогах» в карточке пользователя. После этого нужная переписка сразу откроется на этой вкладке.
