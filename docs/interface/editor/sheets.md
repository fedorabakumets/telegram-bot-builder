# Листы сценария

Листы сценария — это способ разбить большой сценарий бота на отдельные части, чтобы не держать все шаги на одном перегруженном холсте. Каждый лист — это собственное рабочее поле со своими нодами, а переключаться между листами можно через панель листов внизу редактора.

## Зачем нужны листы

Когда бот вырастает и нод становится много, удобнее разложить логику по нескольким листам: например, отдельный лист под приветствие и меню, отдельный — под оформление заказа, ещё один — под поддержку. Так каждый кусок сценария остаётся обозримым, а общая структура — понятной.

## Создание листа

Чтобы добавить новый лист, нажмите кнопку «+» на панели листов. Новый лист получает автоматическое имя «Лист N», где N на единицу больше самого большого из уже существующих номеров листов. Например, если в проекте уже есть «Лист 1» и «Лист 3», следующий новый лист будет назван «Лист 4».

## Переименование листа

Чтобы дать листу понятное название, дважды кликните по его имени на панели листов. Название станет доступным для редактирования — введите новое и подтвердите ввод. Осмысленные имена («Приветствие», «Оплата», «Поддержка») помогают быстрее находить нужную часть сценария.

## Дублирование листа

Лист можно продублировать — при этом создаётся его полная копия вместе со всем содержимым: все ноды и связи между ними переносятся в новый лист. Это удобно, когда нужна похожая ветка сценария: проще скопировать готовый лист и поправить детали, чем собирать всё заново.

## Удаление листа

Удалить лист можно только тогда, когда в проекте больше одного листа. Если лист в проекте остался единственным, удаление недоступно — проект всегда сохраняет хотя бы один лист, и убрать последний нельзя.

## Навигация на мобильных устройствах

На мобильных устройствах переключаться между листами можно жестами-свайпами:

- **Свайп влево** открывает следующий лист.
- **Свайп вправо** открывает предыдущий лист.

На крайних листах свайп за границу ничего не меняет: на первом листе свайп вправо оставляет вас на текущем листе, а на последнем листе свайп влево также не переключает лист.
