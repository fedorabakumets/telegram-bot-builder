# Холст

Холст — это рабочее поле редактора, где из отдельных нод собирается логика бота. Здесь вы добавляете блоки, расставляете их, перемещаете и соединяете между собой, наглядно видя весь сценарий целиком и связи между его шагами.

## Действия с нодами

На холсте доступны четыре основных действия с нодами:

- **Добавление.** Перетащите карточку нужного блока из палитры на холст или нажмите кнопку «+», которая появляется на карточке при наведении.
- **Перемещение.** Захватите ноду и перетащите её в нужное место — так удобно выстраивать порядок шагов.
- **Выделение.** Нажмите на ноду одиночным кликом, чтобы выбрать её. У выбранной ноды открываются настройки в панели свойств.
- **Удаление.** Выделите ноду и нажмите клавишу `Delete`, чтобы убрать её с холста.

## Масштаб и навигация

Холст можно приближать и отдалять в диапазоне от 1 % до 200 %. По умолчанию используется масштаб 100 %. Чтобы изменить его прямо на холсте, удерживайте `Ctrl` и крутите колесо мыши.

Для перемещения по полю удерживайте `Alt` и тяните левой кнопкой мыши (`Alt`+ЛКМ) — холст будет сдвигаться вслед за курсором.

Когда сценарий не помещается на экран, помогает авто-вписывание. Доступны два режима:

- **«Уместить в экран»** — подбирает масштаб так, чтобы текущий вид целиком уместился в видимой области.
- **«Уместить всё»** — отдаляет и центрирует холст так, чтобы стали видны все ноды сценария сразу.

> **Мини-карты на холсте нет.** Чтобы ориентироваться в большом сценарии, перемещайтесь по полю панорамированием (`Alt`+ЛКМ) и используйте режим «уместить всё», который показывает все ноды разом.

## Соединения между нодами

Ноды связываются между собой линиями-соединениями. Цвет линии и её стиль подсказывают, какого типа переход она обозначает.

| Цвет | Назначение | Стиль линии |
|------|------------|-------------|
| 🟡 Жёлтый | Переход из триггера команды к первой ноде | сплошная |
| 🟢 Зелёный | Автоматический переход к следующей ноде | пунктир |
| 🔵 Синий | Переход по нажатию кнопки | пунктир |
| 🟠 Янтарный | Привязка клавиатуры к сообщению | пунктир |
| 🟣 Фиолетовый | Переход после ввода пользователя | пунктир |

> Привязка клавиатуры (🟠 янтарная линия) возможна только в направлении «сообщение → клавиатура».

### Как создать соединение

Наведите курсор на ноду — справа от неё появится выходной порт в виде кружка. Потяните от этого кружка к целевой ноде и отпустите — между нодами создастся соединение.

### Как удалить соединение

Наведите курсор на линию соединения — на ней появится кнопка-крестик. Нажмите на неё, чтобы убрать связь между нодами.
