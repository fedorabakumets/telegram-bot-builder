# Панель инструментов

Панель инструментов — это ряд кнопок над холстом, который собирает в одном месте всё, что нужно для повседневной работы: управление масштабом, отмену и повтор действий, историю изменений, сохранение, копирование и вставку блоков, а также показ или скрытие соседних панелей. Здесь же приведена полная таблица горячих клавиш, чтобы выполнять частые операции быстрее.

## Управление масштабом

В панели есть отдельные кнопки, чтобы уменьшить и увеличить масштаб холста. Рядом доступен выбор конкретного процента: можно тянуть ползунок или выбрать готовое значение из набора (1, 5, 10, 25, 50, 75, 100, 125, 150 и 200). Масштаб задаётся в диапазоне от 1 % до 200 %, значение по умолчанию — 100 %.

Кнопка сброса вида возвращает масштаб и положение холста к исходному состоянию. Кнопки «уместить всё» и «в экран» подбирают масштаб так, чтобы в видимую область поместился весь сценарий целиком.

Переключатель «Авто-уместить при смене листа» (кнопка «A») включает автоматическую подгонку масштаба каждый раз, когда вы открываете другой лист. Состояние переключателя запоминается, поэтому выбранный режим сохраняется между переходами.

## Отмена, повтор и история действий

Кнопки отмены и повтора возвращают холст на шаг назад или вперёд. Отмена убирает последнее изменение, повтор возвращает только что отменённое действие.

Кнопка истории открывает список выполненных действий — добавление, удаление, перемещение, изменение, соединение, разъединение и дублирование блоков. У каждого пункта есть флажок: отметьте нужные действия и откатите именно их, не трогая остальные.

Кнопка сохранения фиксирует текущее состояние проекта.

## Копирование и вставка

Кнопка «Копировать» доступна, когда выделена нода: она помещает выбранный блок в буфер. Кнопка «Вставить» появляется, когда в буфере есть данные, и добавляет блок из буфера на холст. Через буфер удобно переносить ноды между проектами.

## Переключатель «Холст / JSON»

Переключатель «Холст / JSON» меняет вид рабочей области. В режиме «Холст» вы работаете со сценарием визуально, а в режиме «JSON» видите тот же проект в виде структурированного текста.

## Управление видимостью панелей

Панель инструментов позволяет показывать и скрывать соседние области интерфейса: шапку (верхний заголовок), боковую панель и панель свойств. Скрыв ненужные панели, вы освобождаете больше места под холст, а затем теми же кнопками возвращаете их обратно.

## Горячие клавиши

В панели есть справка по горячим клавишам. Полный набор сочетаний приведён в таблице ниже.

| Группа | Действие | Сочетание клавиш |
|--------|----------|------------------|
| Масштаб | Увеличить | `Ctrl` + `+` |
| Масштаб | Уменьшить | `Ctrl` + `-` |
| Масштаб | Сбросить | `Ctrl` + `0` |
| Масштаб | Уместить всё | `Ctrl` + `1` |
| Масштаб | Авто-уместить | `Ctrl` + `2` |
| Редактирование | Удалить узел | `Delete` |
| Редактирование | Дублировать | `Ctrl` + `C` / `D` |
| Редактирование | Копировать в буфер | `Ctrl` + `Shift` + `C` |
| Редактирование | Вставить из буфера | `Ctrl` + `V` |
| История | Отменить | `Ctrl` + `Z` |
| История | Повторить | `Ctrl` + `Y` / `Ctrl` + `Shift` + `Z` |
| Навигация | Масштаб | `Ctrl` + колесо мыши |
| Навигация | Панорамирование | `Alt` + ЛКМ |
| Файл | Сохранить | `Ctrl` + `S` |

> Сочетания с буквами работают и при русской раскладке клавиатуры — например, тем же буквам соответствуют `С`, `М`, `В`. Переключать раскладку перед нажатием не нужно.
