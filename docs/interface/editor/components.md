# Панель компонентов

Панель компонентов — это палитра слева, откуда вы берёте готовые блоки (ноды) и переносите их на холст. Блоки сгруппированы по категориям, чтобы нужный тип находился быстро, а из секции «Команды» можно сразу добавить заготовку реакции на команду. Здесь же находится переключение между палитрой и списком проектов.

## Две вкладки

В левой панели есть две вкладки:

- **Элементы** — палитра блоков, которые можно добавить на холст. Эта вкладка открыта по умолчанию.
- **Проекты** — список ваших проектов и листов сценария.

Чтобы вернуться к палитре после работы со списком проектов, снова откройте вкладку «Элементы».

## Категории нод

На вкладке «Элементы» блоки разложены по шести категориям. Каждая категория группирует связанные по смыслу типы нод и сворачивается, а рядом с названием показан счётчик блоков внутри. Категории идут в таком порядке:

1. **Сообщения** — отправка текста и медиа, ожидание ответа, редактирование и удаление сообщений.
2. **Клавиатура** — кнопки под сообщением и реакции на их нажатие.
3. **Группы** — работа в группах: темы форума, управление участниками.
4. **Автоматизация** — запуск действий по расписанию.
5. **Интеграции** — запросы к внешним сервисам, базы данных, таблицы, условия, переменные, циклы.
6. **Юзербот** — действия от имени пользовательского аккаунта.

## Как добавить ноду

Добавить блок на холст можно двумя способами:

- **Перетаскиванием** — захватите карточку ноды в палитре и перетащите её на нужное место холста.
- **Кнопкой «+»** — наведите курсор на карточку, нажмите появившуюся кнопку «+», и блок добавится на холст.

Перетаскивание работает и на сенсорных экранах: карточку можно перенести на холст касанием и движением пальца.

## Пресеты команд

В секции **«Команды»** собраны готовые заготовки для частых команд бота. Доступно ровно пять пресетов:

- `/start` — приветствие при первом запуске бота.
- `/help` — подсказка о возможностях бота.
- `/settings` — переход к настройкам.
- `/menu` — вызов главного меню.
- **Кастомная команда** — команда с произвольным названием, которое вы задаёте сами.

Каждый пресет добавляет на холст сразу **пару связанных нод** — триггер команды и сообщение, — чтобы не собирать эту связку вручную. Пресеты команд добавляются только перетаскиванием карточки на холст.
