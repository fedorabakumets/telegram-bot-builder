# 📊 Вкладка «Аналитика»

Вкладка «Аналитика» показывает, как растёт аудитория вашего бота и насколько активно люди им пользуются. Здесь собраны наглядные карточки с графиками и круговые диаграммы: по ним видно, сколько у бота пользователей, как меняется их число со временем, сколько идёт сообщений и из каких источников приходят новые люди.

## Всего пользователей

Карточка «Всего пользователей» показывает общее число людей, которые написали боту, и график прироста аудитории под этим числом. По графику видно, как менялось количество пользователей со временем. Если за последнюю неделю появились новые люди, рядом отображается подпись вида «+N за неделю» — это сколько пользователей прибавилось.

## Активность

Карточка «Активность» отражает поток сообщений между ботом и пользователями. У неё есть переключатель режима: можно смотреть все сообщения вместе либо разделить их на входящие (что пишут боту) и исходящие (что отправляет бот). Рядом с числом так же показывается прибавка за неделю и среднее число сообщений на одного пользователя.

## Детализация и вид графика

У каждого графика есть набор переключателей, которые меняют то, как выглядят данные.

- **Детализация по времени** — выбор шага на графике: «1ч», «3ч», «24ч», «30д», «12н», «12м». Чем меньше шаг, тем подробнее видна картина за недавнее время; чем больше — тем длиннее охватываемый период.
- **Режим отображения** — «За период» показывает значения по каждому отрезку времени отдельно, а «Накопительно» складывает их нарастающим итогом, чтобы видеть общий рост.
- **Тип графика** — «Столбчатый» рисует данные столбиками, «Линейный» — линией. Выбирайте тот вид, который вам удобнее читать.

## Диаграммы распределения

Ниже карточек расположены круговые диаграммы, которые показывают, из чего складывается ваша аудитория.

- **Источники трафика** — откуда приходят пользователи: по каким ссылкам и каналам они попадают к боту.
- **Premium** — соотношение пользователей с подпиской Telegram Premium и обычных пользователей (доли «Premium» и «Обычные»).
- **Языки** — на каких языках общаются пользователи бота.

## Выбор бота, проекта и обновление

В шапке вкладки можно выбрать, по какому боту (токену) и какому проекту показывать статистику — это удобно, когда в проекте несколько ботов или у вас несколько проектов. Кнопка «Обновить» перезагружает данные, чтобы увидеть свежие цифры.
