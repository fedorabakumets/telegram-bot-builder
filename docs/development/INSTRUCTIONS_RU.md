# Инструкция по настройке и запуску проекта

Это руководство поможет вам настроить и запустить проект на вашем локальном компьютере с Windows.

## 1. Установка и настройка PostgreSQL

1.  **Установите PostgreSQL** с официального сайта: [https://www.postgresql.org/download/](https://www.postgresql.org/download/)
2.  Во время установки запомните пароль, который вы задаете для пользователя `postgres`.
3.  После установки откройте `psql` (SQL Shell). Он может попросить вас ввести данные для подключения.
    *   Server [localhost]: (нажмите Enter)
    *   Database [postgres]: (нажмите Enter)
    *   Port [5432]: (нажмите Enter)
    *   Username [postgres]: (нажмите Enter)
    *   Password for user postgres: (введите пароль, который вы задали при установке)
4.  В открывшейся командной строке psql выполните команду для создания базы данных:
    ```sql
    CREATE DATABASE telegram_bot_builder;
    ```
5.  Выйдите из psql, введя команду `\q` и нажав Enter.

## 2. Настройка переменных окружения

1.  В корневой папке проекта найдите и запустите файл `create-env.bat`. Этот скрипт создаст файл `.env` на основе `.env.example`.
2.  Откройте созданный файл `.env` в любом текстовом редакторе.
3.  Найдите строку `PGPASSWORD=` и после знака `=` укажите пароль от вашей базы данных PostgreSQL, который вы задали в шаге 1.2. Например: `PGPASSWORD=mysecretpassword`
4.  Убедитесь, что остальные данные для подключения к БД (`PGHOST`, `PGPORT`, `PGDATABASE`, `PGUSER`) соответствуют вашим настройкам. Стандартные значения обычно подходят.

## 3. Установка зависимостей и миграция базы данных

1.  Откройте терминал (командную строку или PowerShell) в папке с проектом.
2.  Выполните команду для установки всех необходимых пакетов:
    ```bash
    npm install
    ```
3.  После завершения установки выполните команду для применения схемы базы данных:
    ```bash
    npm run db:push
    ```
    Эта команда синхронизирует структуру таблиц в вашей базе данных `telegram_bot_builder` с тем, что определено в коде проекта.

## 4. Запуск проекта

1.  Для запуска сервера в режиме разработки используйте скрипт `start-dev.bat`. Просто дважды кликните по нему или выполните в терминале:
    ```bash
    ./start-dev.bat
    ```
2.  После запуска вы должны увидеть в консоли сообщение о том, что сервер запущен, например:
    ```
    10:19:42 PM [express] serving on http://localhost:5000
    ```
3.  Теперь вы можете открыть ваш браузер и перейти по адресу [http://localhost:5000](http://localhost:5000), чтобы увидеть работающее приложение.
