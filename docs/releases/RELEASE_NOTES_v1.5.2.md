## 🎯 Основные изменения

### 🗄️ Полная переработка панели базы данных пользователей

Масштабный рефакторинг системы работы с базой данных пользователей для улучшения архитектуры и поддерживаемости.

#### Новая структура модулей:

- **`client/components/editor/database/user-database/`** — основной модуль панели БД
- **`client/components/editor/database/user-details/`** — панель деталей пользователя
- **`client/components/editor/database/responses-table/`** — таблица ответов пользователей
- **`client/components/editor/database/dialog/`** — диалоговое окно сообщений

#### Компоненты панели пользователей:

- **Desktop/Tablet/Mobile** — адаптивные компоненты для всех устройств
- **DesktopTableRow** — строка таблицы с ячейками данных
- **DesktopLastMessageCell** — ячейка последнего сообщения
- **DesktopUserCell** — ячейка информации о пользователе
- **MobileUserCard** — карточка пользователя для мобильных
- **StatsCards** — карточки статистики

#### Компоненты деталей пользователя:

- **UserDetailsDialog** — диалог с подробной информацией
- **UserInfoGrid** — сетка основной информации
- **UserDates** — даты активности
- **UserTags** — теги пользователя
- **UserStatus** — переключатель статуса
- **RawJson** — просмотр сырых данных JSON
- **ResponseCard** — карточка ответа
- **ResponseMedia** — медиа в ответах

#### Таблица ответов:

- **ResponsesTableWithPagination** — таблица с пагинацией
- **ResponseRow** — строка ответа
- **ResponsePhoto** — просмотр фото в ответах
- **UserResponsesList** — список ответов пользователя
- **UserResponsesPreview** — превью ответов
- **PaginationControls** — управление пагинацией
- **ItemsPerPageSelector** — выбор количества элементов
- **ResponsesUserFilter** — фильтр по пользователю
- **ResponseCount** — счётчик ответов

#### Диалог сообщений:

- **MessageDialog** — диалог переписки
- **MessageBubble** — пузырь сообщения
- **MessageButtons** — кнопки сообщения
- **MessageMedia** — медиа вложения
- **MessageText** — текст сообщения
- **MessageTimestamp** — время сообщения
- **MessageAvatar** — аватар отправителя
- **FormattedText** — форматированный текст
- **NodeSender** — отправка узла

#### Хуки и утилиты:

- **useUsers** — получение списка пользователей
- **useUserDetails** — получение деталей пользователя
- **useDialogMessages** — получение сообщений диалога
- **useLastMessage** — получение последнего сообщения
- **useSearchUsers** — поиск пользователей
- **useStats** — получение статистики
- **useSendMessage** — отправка сообщения
- **useUpdateUser** — обновление пользователя
- **useDeleteUser** — удаление пользователя
- **useDeleteAllUsers** — удаление всех пользователей
- **parseResponseData** — парсинг данных ответов
- **formatUserName** — форматирование имени пользователя
- **formatRelativeTime** — форматирование относительного времени

### ✏️ Rich Text редактор для сообщений

Новый компонент редактирования текста с поддержкой форматирования и переменных.

#### Возможности:

- **Форматирование текста** — жирный, курсив, подчёркивание, зачёркивание
- **Вставка переменных** — {{variable}} из контекста бота
- **Фильтрация переменных** — поиск и фильтрация доступных переменных
- **Статистика текста** — подсчёт символов, слов, строк
- **Буфер обмена** — копирование/вставка с сохранением форматирования
- **Undo/Redo** — отмена и повтор действий
- **HTML конвертация** — преобразование между HTML и внутренним форматом

#### Компоненты:

- **InlineRichEditor** — основной компонент редактора
- **CompactInlineEditor** — компактная версия
- **EditorToolbar** — панель инструментов форматирования
- **CompactToolbar** — компактная панель
- **VariableFilterMenu** — меню фильтра переменных
- **VariableFilterToggle** — переключатель фильтра
- **VariablesMenu** — меню переменных
- **VariableMenuItem** — элемент меню переменной
- **UsedVariablesList** — список использованных переменных
- **FormattedText** — отображение форматированного текста
- **StatsBar** — панель статистики текста
- **EditorContent** — содержимое редактора

#### Хуки:

- **useInlineRichEditor** — основной хук редактора
- **useFormatting** — управление форматированием
- **useVariableInsert** — вставка переменных
- **useUndoRedo** — отмена/повтор
- **useClipboard** — работа с буфером обмена
- **useTextStats** — статистика текста
- **useEditorSync** — синхронизация редактора
- **useEditorInput** — обработка ввода
- **useKeyboardShortcuts** — горячие клавиши

#### Утилиты:

- **formatOptions** — опции форматирования
- **formattingParser** — парсер форматирования
- **htmlConverter** — конвертация HTML
- **variableDisplayUtils** — отображение переменных

### 🔧 Исправление ошибки ReferenceError indentLevel

Исправлена критическая ошибка в генераторе кода.

#### Проблема:

При генерации Python-кода возникала ошибка `ReferenceError: indentLevel is not defined` в функции `generateHandleNodeFunctions`.

#### Решение:

- Исправлена область видимости переменной `indentLevel`
- Переменная объявлена в правильном контексте

#### Файлы:

- **`client/lib/generate/generateHandleNodeFunctions.ts`** — исправлена ошибка

## 🚀 Установка

### Обновление с v1.5.1:

```bash
# Обновите проект
git pull origin main

# Установите зависимости
cd client
npm install

# Соберите проект
npm run build

# Запустите
npm run start
```

### Через Docker:

```bash
# Пересоберите образ
docker-compose build --no-cache

# Запустите
docker-compose up -d
```

### Обновление сгенерированных ботов:

```bash
# В сгенерированном боте установите зависимости
cd generated_bot
pip install -r requirements.txt

# python-dotenv теперь устанавливается автоматически
```
