# 📦 Telegram Bot Builder v2.0.0

**Дата выпуска:** 10 марта 2026 г.

**Сравнение версий:** [v1.5.2...v2.0.0](https://github.com/fedorabakumets/telegram-bot-builder/compare/v1.5.2...v2.0.0)

---

## 🎯 Основные изменения

### 🎨 Обновлённая шапка сайта

Масштабное улучшение шапки веб-сайта для улучшения UX и визуального стиля.

#### Изменения:

- **Убрана авторизация** — компонент авторизации Telegram исключён из шапки
- **Убраны кнопки управления состояниями** — скрыты переключатели панелей
- **Убран переключатель темы** — переключатель темы оформления удалён
- **Переименование кнопок** — «Сохранить шаблон» → «Сохранить сценарий», «Шаблоны» → «Сценарии»
- **Добавлен бейдж версии** — отображение текущей версии приложения (v2.0.0)
- **Перемещена кнопка GitHub** — компактная иконка рядом с версией

#### Компоненты:

- **VersionBadge** — новый компонент бейджа версии в цветах бренда
- **GithubButton** — обновлённая компактная кнопка (только иконка)
- **BrandSection** — обновлённая секция бренда с версией и GitHub

---

### 📑 Вкладка «Редактор»

#### Панель свойств

Оптимизация панели свойств для улучшения рабочего процесса.

##### Изменения:

- **Скрытие секций** — все секции кроме «Текст сообщения» скрыты по умолчанию
- **Отображение ответов** — выбор между списком или через запятую в секции «Текст сообщения»
- **Редактирование кнопок** — возможность редактировать расположение кнопок в секции «Клавиатура»
- **Сбор ответов** — обновлённая секция с новыми возможностями:
  - Сохранение ответов в одной переменной на нескольких узлах
  - Опция перезаписи данных или сохранения в массиве
- **Убран переключатель «Сохранить в CSV»** — функция удалена
- **Секция обновлений** — добавлены надписи «Скоро обновление»
- **Прикреплённый медиафайл** — улучшенная секция загрузки:
  - Улучшена загрузка по URL
  - Улучшена загрузка с устройства

---

### 🤖 Вкладка «Бот»

#### Разделение пространства:

- **Список ботов** — отдельная панель со списком всех ботов
- **Терминал** — отдельная панель для логов и вывода

#### Новые возможности:

- **Аватарки ботов** — отображение аватарок для каждого бота
- **Улучшенная навигация** — быстрое переключение между списком и терминалом

---

### 💻 Вкладка «Код»

#### Временные изменения:

- **Экспорт в Google Таблицы** — временно недоступен

---

### 👥 Вкладка «Пользователи»

Масштабная работа по улучшению работы с пользователями.

#### Улучшения:

- **Аватары** — корректное отображение аватаров ботов и пользователей
- **Таблица ответов** — улучшенная работа с ответами:
  - Сортировка по пользователям
  - Исправлено отображение ответа при отсутствии файла
  - Улучшено отображение последнего времени активности и сообщений
  - Удалены ненужные кнопки (детали пользователя, диалог, активация)
- **Экспорт в Google Таблицы** — временно отключён
- **Форматирование текста** — добавлена возможность форматирования для рассылок
- **Рассылка узлов** — теперь можно рассылать узлы:
  - Медиа и автопереходы пока недоступны
  - Альтернатива — рассылка через сценарий
- **Селекторы пользователей** — добавлены в панелях диалога и деталях

---

### 🔧 Исправление команды разработки

Восстановлена работоспособность команды запуска режима разработки.

#### Изменения:

- **`npm run dev`** — команда теперь работает корректно
- **Vite dev server** — исправлены проблемы с запуском
- **Hot Module Replacement** — горячая перезагрузка работает стабильно

#### Использование:

```bash
# Запуск режима разработки
npm run dev

# Сервер доступен по адресу
http://localhost:5173
```

- **Кнопка удаления ID** — добавлена кнопка удаления напротив каждого ID

## 🚀 Установка

### Обновление с v1.5.2:

```bash
# Обновите проект
git pull origin main

# Установите зависимости
cd client
npm install

# Соберите проект
npm run build

# Запустите
npm run start
```

### Через Docker:

```bash
# Пересоберите образ
docker-compose build --no-cache

# Запустите
docker-compose up -d
```

## ⚠️ Важные замечания

### Временные ограничения

- **Экспорт в Google Таблицы** — временно недоступен в вкладках «Код» и «Пользователи»


### Переименование

- Все упоминания «шаблон» в интерфейсе заменены на «сценарий»
- Кнопки «Сохранить шаблон» и «Шаблоны» переименованы

### Панель свойств

- Секции скрыты по умолчанию для улучшения фокуса
- Открывайте только необходимые секции для работы
