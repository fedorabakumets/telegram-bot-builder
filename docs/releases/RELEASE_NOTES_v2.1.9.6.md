# 📋 Release Notes v2.1.9.6

**Дата выпуска:** 19 мая 2026 г.

---

## 🔥 Главное в этом релизе

### Полная миграция генерации кода на Jinja2

Генерация Python-кода полностью переведена с TypeScript-конкатенации на систему Jinja2/Nunjucks шаблонов. 90+ шаблонов с типизированными параметрами (Zod-схемы), рендерерами и фазовыми тестами.

---

## 🧩 Новые узлы (ноды)

| Узел | Описание |
|------|----------|
| **text_trigger** | Текстовый триггер — реакция на произвольный текст |
| **command_trigger** | Командный триггер — отдельный узел для /команд |
| **forward_message** | Пересылка сообщений в другой чат |
| **schedule_trigger** | Триггер по расписанию (cron) |
| **loop** | Цикл — повторение действий |
| **map** | Маппинг — преобразование данных |
| **delay** | Задержка между действиями |
| **set_variable** | Установка переменной |
| **psql_query** | SQL-запрос к PostgreSQL |
| **http_request** | HTTP-запрос к внешнему API |
| **convert_file** | Конвертация файлов |
| **create_forum_topic** | Создание темы в форуме |
| **edit_message** | Редактирование отправленного сообщения |
| **condition** | Условное ветвление |

## 🚀 Новые возможности

- **Рассылки (Broadcasts)** — полный модуль: создание, preview аудитории, отправка, остановка
- **Таблицы данных** — CRUD для пользовательских таблиц (колонки, строки, переиндексация)
- **Коллабораторы** — добавление/удаление участников проекта
- **Переменные окружения** — управление env-переменными через UI
- **Прокси для Telegram API** — поддержка HTTP/SOCKS прокси
- **Динамические кнопки** — генерация кнопок из переменных
- **Проверка подписки на канал** — условие в сценарии
- **Серверная генерация кода** — перенос генерации с клиента на сервер
- **Применение изменений к запущенному боту** — без перезапуска
- **Redis fallback** — автоматический переход на PostgreSQL при недоступности Redis
- **Content protection** — защита контента per-bot
- **Поиск по сообщениям** в диалогах
