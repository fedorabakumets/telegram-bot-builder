# agent-tokens

Эндпоинтов: **3**

### `GET` /api/agent-tokens

Список токенов агента

**Авторизация:** Cookie (`connect.sid`)

Требует session cookie. Возвращает PAT текущего пользователя без секрета (только prefix и метаданные).

#### Ответы

| Код | Описание |
|-----|----------|
| 200 | Массив токенов владельца |
| 401 | Не авторизован |

### `POST` /api/agent-tokens

Создать токен агента

**Авторизация:** Cookie (`connect.sid`)

Требует session cookie. Полный секрет `token` возвращается один раз — сохраните сразу.

**Тело запроса:** `CreateAgentTokenRequest`

#### Ответы

| Код | Описание |
|-----|----------|
| 201 | Токен создан |
| 400 | Ошибка валидации |
| 401 | Не авторизован |

### `DELETE` /api/agent-tokens/{id}

Отозвать токен агента

**Авторизация:** Cookie (`connect.sid`)

Требует session cookie. Отзывает только токен текущего пользователя.

**Параметры:** 1

#### Ответы

| Код | Описание |
|-----|----------|
| 200 | Токен отозван |
| 401 | Не авторизован |
| 404 | Токен не найден или чужой |
