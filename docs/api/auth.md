### `POST` /api/auth/telegram

Вход через Telegram Login Widget

**Авторизация:** Публичный

Публичный. Создаёт/обновляет пользователя и устанавливает session cookie.

**Тело запроса:** `TelegramAuthRequest`

#### Ответы

| Код | Описание |
|-----|----------|
| 200 | Авторизация успешна, cookie установлена |
| 400 | Не передан id |
| 401 | Невалидный id_token |

### `GET` /api/auth/telegram/user/{id}

Получить пользователя Telegram по ID

**Авторизация:** Публичный

Публичный. Возвращает данные из таблицы telegram_users.

**Параметры:** 1

#### Ответы

| Код | Описание |
|-----|----------|
| 200 | Пользователь найден |
| 400 | Невалидный id |
| 404 | Пользователь не найден |
