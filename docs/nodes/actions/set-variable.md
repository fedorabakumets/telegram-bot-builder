# Установить переменную (set_variable)

Сохраняет значение в пользовательскую переменную. Переменные хранятся per-user — у каждого пользователя свои значения.

## Когда использовать

- Сохранить выбор пользователя (язык, город, тариф)
- Увеличить счётчик (баланс, очки, попытки)
- Пометить пользователя (verified, banned, vip)
- Сохранить результат HTTP-запроса
- Очистить данные (сбросить корзину)

## Настройки

| Поле | Описание |
|------|----------|
| Переменная | Имя переменной для записи |
| Режим | Как записать значение (set, increment, append, toggle) |
| Значение | Что записать (текст, число, переменная, выражение) |
| Автопереход | Куда перейти после сохранения |

## Режимы записи

| Режим | Описание | Пример |
|-------|----------|--------|
| `set` | Записать новое значение | `balance` = `100` |
| `increment` | Прибавить число | `balance` += `50` |
| `decrement` | Вычесть число | `balance` -= `10` |
| `append` | Добавить к строке/массиву | `history` += `"новая запись"` |
| `toggle` | Переключить true/false | `notifications` = !`notifications` |

## Значение

В поле "Значение" можно использовать:

- Статический текст: `hello`
- Число: `100`
- Переменную: `{user_input}`
- Системную переменную: `{user_id}`, `{first_name}`
- Пустую строку (для очистки)

## Примеры

### Установить язык

```json
{
  "type": "set_variable",
  "data": {
    "variable": "language",
    "mode": "set",
    "value": "ru",
    "autoTransitionTo": "menu"
  }
}
```

### Увеличить баланс

```json
{
  "type": "set_variable",
  "data": {
    "variable": "balance",
    "mode": "increment",
    "value": "50",
    "autoTransitionTo": "balance_updated"
  }
}
```

### Сохранить ввод пользователя

```json
{
  "type": "set_variable",
  "data": {
    "variable": "user_city",
    "mode": "set",
    "value": "{user_input}",
    "autoTransitionTo": "confirm_city"
  }
}
```
