# Сообщение (message)

Отправляет текстовое сообщение пользователю. Основная рабочая нода — используется в 90% сценариев.

## Когда использовать

- Приветствие после `/start`
- Меню с кнопками
- Информационные сообщения
- Запрос ввода данных от пользователя
- Отправка медиа (фото, видео, документы)

## Настройки

| Поле | Описание |
|------|----------|
| Текст сообщения | Поддерживает переменные `{имя}` и форматирование |
| Тип клавиатуры | `inline` (под сообщением), `reply` (внизу экрана), `none` |
| Кнопки | Список кнопок с текстом и действием |
| Форматирование | HTML, Markdown или без форматирования |
| Медиа | Фото, видео, аудио, документ (URL или file_id) |
| Автопереход | Перейти к следующей ноде автоматически |
| Только приватные | Работает только в личных сообщениях |
| Только админы | Работает только для администраторов |

## Переменные в тексте

В тексте сообщения можно использовать переменные:

```
Привет, {first_name}! У тебя {balance} монет.
```

Системные переменные: `{user_id}`, `{first_name}`, `{last_name}`, `{username}`, `{message_text}`, `{chat_id}`.

Пользовательские: любые, которые ты сохранил через set_variable или input.

### Фильтры

```
Твои покупки: {items|join:", "}
```

## Типы кнопок

### Inline (под сообщением)

```json
{
  "buttons": [
    { "text": "✅ Да", "action": "goto", "target": "confirm_node" },
    { "text": "❌ Нет", "action": "goto", "target": "cancel_node" }
  ],
  "keyboardType": "inline"
}
```

### Reply (внизу экрана)

```json
{
  "buttons": [
    { "text": "📋 Меню", "action": "goto", "target": "menu" },
    { "text": "⚙️ Настройки", "action": "goto", "target": "settings" }
  ],
  "keyboardType": "reply"
}
```

## Медиа

Добавь URL изображения, видео или документа — бот отправит его вместе с текстом.

Поддерживаются переменные: `{photo_url}` — подставится значение из переменной пользователя.

Несколько медиа — через `attachedMedia` (отправятся как медиа-группа).

## Отправка нескольким получателям

Можно отправить сообщение не только текущему пользователю, но и в другие чаты:

| Тип | Описание |
|-----|----------|
| `user` | Текущий пользователь (по умолчанию) |
| `chat_id` | Конкретный чат по ID или @username |
| `admin_ids` | Все администраторы бота |

## Сохранение ID сообщения

Поле `saveMessageIdTo` — сохраняет ID отправленного сообщения в переменную. Полезно для последующего редактирования через edit_message.

## Примеры

### Простое приветствие с меню

```json
{
  "type": "message",
  "data": {
    "messageText": "Привет, {first_name}! Выбери раздел:",
    "keyboardType": "inline",
    "buttons": [
      { "text": "📖 Каталог", "action": "goto", "target": "catalog" },
      { "text": "🛒 Корзина", "action": "goto", "target": "cart" },
      { "text": "👤 Профиль", "action": "goto", "target": "profile" }
    ]
  }
}
```

### Сообщение с фото

```json
{
  "type": "message",
  "data": {
    "messageText": "Вот наш новый товар!",
    "imageUrl": "https://example.com/product.jpg",
    "keyboardType": "inline",
    "buttons": [
      { "text": "🛒 Купить", "action": "goto", "target": "buy" }
    ]
  }
}
```
