# Триггер команды (command_trigger)

Срабатывает когда пользователь отправляет команду боту — `/start`, `/help` или любую кастомную.

## Когда использовать

- Точка входа в бота (команда `/start`)
- Навигация по разделам (`/menu`, `/help`, `/settings`)
- Админские команды (`/stats`, `/broadcast`)

## Настройки

| Поле | Описание |
|------|----------|
| Команда | Текст команды, например `/start` |
| Описание | Описание для меню бота (видно в списке команд Telegram) |
| Показать в меню | Добавить команду в синюю кнопку "Меню" в Telegram |
| Только приватные чаты | Команда работает только в личных сообщениях с ботом |
| Автопереход | Нода, к которой перейти после срабатывания триггера |

## Deep link параметры

Если пользователь переходит по ссылке `t.me/bot?start=параметр`, значение `параметр` автоматически сохраняется в переменную `{deep_link_param}`.

Реферальные ссылки: если параметр начинается с `ref_` (например `t.me/bot?start=ref_12345`), то `{referrer_id}` = `12345`.

## Примеры

### Простой стартовый триггер

```json
{
  "type": "command_trigger",
  "data": {
    "command": "/start",
    "description": "Запустить бота",
    "showInMenu": true,
    "autoTransitionTo": "welcome_message"
  }
}
```

### Команда только для приватных чатов

```json
{
  "type": "command_trigger",
  "data": {
    "command": "/settings",
    "description": "Настройки",
    "isPrivateOnly": true,
    "autoTransitionTo": "settings_menu"
  }
}
```
