# Справочник нод

Ноды — строительные блоки бота. Каждая нода выполняет одну задачу: принимает событие, отправляет сообщение, проверяет условие или собирает данные.

## Категории

| Категория | Описание |
|-----------|----------|
| [📨 Триггеры](./triggers/) | Точки входа — что запускает сценарий |
| [⚡ Действия](./actions/) | Что бот делает — отправка, запросы, сохранение |
| [🔀 Логика](./logic/) | Управление потоком — условия, задержки, циклы |
| [📝 Ввод данных](./input/) | Сбор информации от пользователя |

## Как ноды соединяются

1. **Триггер** запускает сценарий (команда, текст, событие)
2. **Действие** выполняет работу (отправка, сохранение)
3. **Кнопки** или **автопереход** ведут к следующей ноде
4. **Логика** управляет ветвлением (условия, циклы)
5. **Узел handle_user_input** останавливает поток и ждёт ответа пользователя, сохраняя его в переменную

## Общие поля

Эти поля есть у большинства нод:

| Поле | Описание |
|------|----------|
| `autoTransitionTo` | Автоматический переход к следующей ноде |
| `isPrivateOnly` | Работает только в личных сообщениях |
| `adminOnly` | Работает только для администраторов |
| `buttons` | Кнопки (inline или reply) |
| `keyboardType` | Тип клавиатуры: inline, reply, none |
