# Шаблон fake-callback

## Описание

Генерирует Python-код создания `fake_callback` для автоперехода между узлами. Используется когда нужно имитировать нажатие кнопки для перехода к следующему узлу.

## Параметры

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|----------|-----|--------------|--------------|----------|
| `targetNodeId` | `string` | Да | — | ID целевого узла |
| `sourceNodeId` | `string` | Да | — | ID исходного узла |
| `safeFunctionName` | `string` | Да | — | Безопасное имя функции (без спецсимволов) |
| `indentLevel` | `string` | Нет | `'    '` | Базовый отступ |

## Тесты

```bash
npm run test:fake-callback
```
