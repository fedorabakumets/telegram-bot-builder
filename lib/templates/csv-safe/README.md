# Шаблон csv-safe

## Описание

Генерирует Python-код для безопасной работы с CSV файлами: запись и чтение с обработкой ошибок.

## Параметры

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|----------|-----|--------------|--------------|----------|
| `operation` | `'write' \| 'read'` | Да | — | Операция |
| `csvFileVar` | `string` | Да | — | Имя переменной с путём к файлу |
| `dataVar` | `string` | Нет | — | Имя переменной с данными (для write) |
| `resultVar` | `string` | Нет | — | Имя переменной для результата (для read) |
| `indentLevel` | `string` | Нет | `'                '` | Базовый отступ |

## Тесты

```bash
npm run test:csv-safe
```
