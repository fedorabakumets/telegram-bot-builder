# Шаблон media-path-resolve

## Описание

Генерирует Python-код для резолвинга пути к медиафайлу: проверяет, является ли URL локальным путём `/uploads/...` и создаёт `FSInputFile`.

## Параметры

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|----------|-----|--------------|--------------|----------|
| `mediaType` | `'photo' \| 'video' \| 'audio' \| 'document'` | Да | — | Тип медиа |
| `urlVar` | `string` | Да | — | Имя переменной с URL |
| `indentLevel` | `string` | Нет | `'                '` | Базовый отступ |

## Тесты

```bash
npm run test:media-path-resolve
```
