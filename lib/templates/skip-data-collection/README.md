# Шаблон skip-data-collection

## Описание

Генерирует Python-код проверки флага `skipDataCollection`. Если переход был через кнопку с `skipDataCollection`, переменная не сохраняется.

## Параметры

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|----------|-----|--------------|--------------|----------|
| `variableName` | `string` | Да | — | Имя переменной для сохранения |
| `variableValue` | `string` | Да | — | Выражение значения переменной |
| `indentLevel` | `string` | Нет | `'    '` | Базовый отступ |

## Тесты

```bash
npm run test:skip-data-collection
```
