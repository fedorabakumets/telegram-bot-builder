# Шаблон conditional-messages

## Описание

Генерирует Python-код для выбора текста сообщения на основе условий (данных пользователя).

## Параметры

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|----------|-----|--------------|--------------|----------|
| `conditionalMessages` | `ConditionalMessage[]` | Да | — | Массив условных сообщений |
| `defaultText` | `string` | Да | — | Текст по умолчанию (fallback) |
| `indentLevel` | `string` | Нет | `'                '` | Базовый отступ |

## Тесты

```bash
npm run test:conditional-messages
```
